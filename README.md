# fiscoabierto.org

Sitio web de Fisco Abierto: herramienta fiscal gratuita de utilidad pública para la economía digital mexicana.

## Stack
- HTML / CSS / JS puro (sin frameworks)
- Hosting: Netlify (free tier)
- Dominio: fiscoabierto.org (Namecheap)

## Estructura
```
web/
├── index.html    # Sitio completo (single-page)
├── .gitignore
└── README.md
```

## Deploy
El sitio se despliega automáticamente en Netlify al hacer cambios.
